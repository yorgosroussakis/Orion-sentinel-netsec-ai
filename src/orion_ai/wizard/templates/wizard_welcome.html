Security & Monitoring Setup Wizard

Welcome to Your Security Pi!

This wizard will help you configure Orion Sentinel - your network security monitoring and AI-powered threat detection system.

What This Pi Does:

  • Network Security Monitoring: Passive IDS with Suricata on mirrored traffic
  • AI Detection: Anomaly detection and domain risk scoring
  • Threat Intelligence: Real-time IOC feeds and correlation
  • Automated Response: SOAR-lite playbooks and actions
  • Visualization: Grafana dashboards for security insights

Setup Steps:

  1. Connect to your DNS Pi (Pi #1)
  2. Configure network interface and security mode
  3. Enable AI detection and threat intelligence
  4. Review and apply configuration

Note: This setup takes about 3-5 minutes. You can always adjust settings later through configuration files.
